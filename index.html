Paraglider Tools – MCP Server for AI agent integrations

Endpoint
https://para-tools.uk/para-mcp/mcp

{
  "servers": {
    "paraglider-tools": {
      "type": "streamable-http",
      "url": "https://para-tools.uk/para-mcp/mcp"
    }
  }
}
